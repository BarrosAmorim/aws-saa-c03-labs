# AWS SAA-C03 Labs

Este repositório reúne os laboratórios práticos desenvolvidos durante minha preparação para a certificação **AWS Certified Solutions Architect – Associate (SAA-C03)**.

Cada laboratório foi criado com o objetivo de aplicar, na prática, os conceitos estudados, documentando todo o processo de implementação, as decisões tomadas, as boas práticas recomendadas pela AWS e os resultados obtidos.

Além de servir como material de estudo, este repositório funciona como um portfólio técnico, demonstrando a utilização dos principais serviços da AWS em cenários reais de aprendizado.

---

## Objetivos

- Consolidar os conhecimentos adquiridos durante os estudos para a certificação AWS Certified Solutions Architect – Associate (SAA-C03).
- Desenvolver experiência prática por meio da implementação de laboratórios utilizando os principais serviços da AWS.
- Documentar cada projeto de forma clara e organizada, seguindo padrões utilizados em documentação técnica.
- Aplicar boas práticas de segurança, arquitetura, disponibilidade, desempenho e otimização de custos recomendadas pela AWS.
- Construir um portfólio técnico público para demonstrar a evolução dos estudos e das habilidades em computação em nuvem.
